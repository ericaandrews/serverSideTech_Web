The Visual Studio Code website, https://code.visualstudio.com/, offers introductory videos on different features of Visual Studio Code. 
The following are some of the video tutorials and links offered:
 
Getting Started
This tutorial helps with installation and learning the basic features of Visual Studio Code
https://code.visualstudio.com/docs/introvideos/basics
The outline states this tutorial assist with ‘downloading and installing Visual Studio Code’, ‘seeing an overview of the user interface,’ 
‘installing support for your favorite programming languages’, ‘changing your keyboard shortcuts and easily migrating from editors using keybinding extensions’, 
customizing your editor with themes, and exploring Visual Studio Code features in the Interactive Editor Playground’.
 
Interactive Editor Playground - Visual Studio Code
The ‘Interactive Editor Playground’ is located on the welcome page and introduces a step-by-step guide to ‘advanced code editing features with interactive examples’.
The playground offers ‘advanced editing features without creating files or projects’ on Visual Studio Code.
https://code.visualstudio.com/updates/v1_9#_interactive-playground
 
 
Customize
This tutorial helps with personalization through ‘settings, themes, and keybinding’ in Visual Studio Code.
https://code.visualstudio.com/docs/introvideos/configure
The outline states this tutorial assist with ‘changing your UI Color Theme and File Icon Theme’, ‘finding customization options in user and workspace settings’, 
‘learning and changing keyboard shortcuts (key bindings)’, and ‘writing code faster with snippets and creating your own custom snippets’.
 
Snippets - Visual Studio Code
The summary states that ‘code snippets are templates that make it easier to enter repeating code patterns, such as loops or conditional statements.
Many extensions of the Visual Studio Code Marketplace include snippets’. If you find one you like, install the snippet and restart Visual Studio Code. 
After Visual Studio Code restarts the new snippet should be available.
https://code.visualstudio.com/docs/editor/userdefinedsnippets
